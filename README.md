# Week6_Aayush_Kumar_Autoencoder_MNIST
This project implements different Autoencoder architectures for MNIST image reconstruction and denoising using PyTorch.
## Models Implemented
- FFNN Autoencoder
- Transpose CNN Autoencoder
- Upsampled CNN Autoencoder

## Dataset
MNIST Handwritten Digits

## Libraries
- PyTorch
- NumPy
- Matplotlib
- Torchvision

## Results
The models successfully reconstruct handwritten digit images from compressed latent representations. 
CNN-based autoencoders produce better reconstruction quality compared to the fully connected autoencoder.
